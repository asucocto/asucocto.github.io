Projects

This is how we improve students' lives.

Berkeley Mobile

Berkeley Mobile is the campus flagship mobile application made by students, for students. It offers a number of features, including Bear Transit, Dining Hall Menus, Library Hours, Gym Hours, and more

BerkeleyTime

Berkeleytime is a course discovery platform that allows you to figure out everything you need to know about that class you want to take.

Project Hermione

Project Hermione is a Computer Vision based lecture casting service designed for individuals in the DSP program who are in classes where ETS does not perform lecture casting.

Digital Service

The Digital Service is a digital consulting service designed for on-campus entities, especially in the ASUC, who want to have a project built for them. We're always looking for new projects!
